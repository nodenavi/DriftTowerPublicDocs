# Controls


## Controller (Preferred)
![alt text](gamepadLayout.png)

| Button | Description |
| ----------- | ----------- |
| Left Trigger  | brake |
| Right Trigger | accelerate |
| Left Stick Horizontal | steer |
| Button East | shoot |

## Keybaord and Moused (Not Preferred)

| Button | Description |
| ----------- | ----------- |
| s | brake |
| w | accelerate |
| a | steer left |
| d | steer right |
| left click  | shoot |
